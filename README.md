# 小说写作项目

这是一个10万字小说的写作项目，使用Cursor编辑器进行创作。

## 项目结构

- `outline/` - 大纲和剧情结构
- `characters/` - 人物设定和关系
- `worldbuilding/` - 世界观和背景设定
- `chapters/` - 正文章节
- `notes/` - 写作笔记和灵感记录
- `tools/` - 写作辅助工具

## 写作目标

- 总字数：100,000字
- 预计章节：20-30章
- 每章字数：3,000-5,000字

## 使用指南

1. 在开始写作前，完善`outline/`中的大纲
2. 详细设定`characters/`和`worldbuilding/`
3. 按章节在`chapters/`中进行创作
4. 使用`notes/`记录灵感和修改想法
5. 定期使用`tools/`中的脚本检查进度
